# VolumeXP

**VolumeXP** — это легковесный, нативный системный планировщик громкости для операционных систем семейства Windows (с гарантированной обратной совместимостью вплоть до Windows XP SP2). Программа написана на чистом C++ (Win32 API) без использования тяжелых фреймворков и сторонних утилит (таких как nircmd), управляя системным микшером напрямую.

## Основной функционал

*   **Гибкое расписание по дням недели:** Создавайте правила для конкретных дней (например, только для понедельника и среды, или только для выходных), используя удобные чекбоксы.
*   **Правила "На весь день":** Специальная галочка "Весь день" позволяет зафиксировать громкость на полные сутки (сохраняя значение громкости в 23:59 и плавно перенося его на следующий день).
*   **Умная защита от пересечений (Collision Detection):** При добавлении нового правила программа побуквенно анализирует выбранные дни недели и время. Если интервалы наслаиваются друг на друга (даже частично), программа заблокирует создание конфликтного правила.
*   **Громкость по умолчанию:** Когда ни одно из ваших правил по расписанию не активно, VolumeXP автоматически возвращает системный звук на заданную "громкость по умолчанию" (например, 100%).
*   **Кнопка "Применить" (Force Run):** Умная синхронизация звука. Высчитывает, какая громкость должна быть выставлена именно в эту минуту согласно вашей таблице расписаний, и принудительно устанавливает её. Эта же логика срабатывает автоматически при запуске программы (полезно при случайной перезагрузке ПК в середине дня).
*   **Скрытый режим и Автозагрузка:** Возможность работы полностью в фоновом режиме (в системном трее) с автоматическим запуском вместе с Windows.
*   **Журнал событий (Логирование):** При каждом автоматическом срабатывании (изменении громкости по времени) программа делает запись в файл `schedule.log` для последующего контроля.
*   **Защита от двойного запуска (Single Instance):** Программа следит за тем, чтобы в памяти не висело больше одного экземпляра процесса (используя системные Mutex).
*   **Мониторинг громкости в реальном времени:** Программа раз в секунду опрашивает звуковую карту и выводит текущую аппаратную громкость прямо в интерфейсе.

## Установка и запуск

Программа является портативной (Portable) и состоит из одного `.exe` файла.
1. Скачайте `VolumeXP.exe` из раздела **[Releases](https://github.com/mrakoz/VolumeXP/releases)**.
2. Запустите файл. Рядом с ним автоматически создастся конфигурационный файл `settings.ini` (в правильной кодировке UTF-16 LE).
3. Настройте ваше расписание, нажмите "Сохранить", и можете смело сворачивать программу в трей!

## Разработка / Сборка из исходников

Проект компилируется с помощью MinGW (GCC) с флагами строгой обратной совместимости для работы на Windows XP.

**Команда для сборки:**
```bash
g++.exe main.cpp resource.res -o VolumeXP.exe -mwindows -static -lcomctl32 -lwinmm -D_WIN32_WINNT=0x0501 -DWINVER=0x0501
```
